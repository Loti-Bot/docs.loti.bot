### <Icon icon="party-horn" size={32} /> Creating a Giveaway

A giveaway can be created by using`/gstart`.

<CodeGroup>
  ```plaintext Syntax
  /gstart (optional: channel)
  ```

  ```plaintext Example
  /gstart channel:#giveaways
  ```
</CodeGroup>

### <Icon icon="pencil" size={32} /> Editing a Giveaway

<AccordionGroup>
  <Accordion title="End the giveaway early">
    End an active giveaway before its scheduled completion.

    <CodeGroup>
      ```plaintext Syntax
      ,giveaway end (giveaway id)
      ```

      ```plaintext Example
      ,giveaway end 1318805763531083800
      ```
    </CodeGroup>

    * **Permissions**: Manage Guild

    * **Aliases**: stop, finish
  </Accordion>

  <Accordion title="Edit the giveaway">
    Modify details of an active giveaway.

    <CodeGroup>
      ```plaintext Syntax
      ,giveaway edit (giveaway id) [giveaway winners] [giveaway prize]
      ```

      ```plaintext Example
      ,giveaway edit 1318805763531083800 2 Prize
      ```
    </CodeGroup>

    * **Permissions**: Manage Guild
  </Accordion>
</AccordionGroup>

### <Icon icon="gear" size={32} /> Editing the Giveaway Settings

<AccordionGroup>
  <Accordion title="Creating extra entries">
    Set additional entries for users with a specific role.

    <CodeGroup>
      ```plaintext Syntax
      ,giveaway settings extra-entries (role) (extra entries)
      ```

      ```plaintext Example
      ,giveaway settings extra-entries @member 3
      ```
    </CodeGroup>

    * **Permissions**: Manage Guild
  </Accordion>

  <Accordion title="Removing extra entries">
    Remove extra entries from users with a specific role.

    <CodeGroup>
      ```plaintext Syntax
      ,giveaway settings remove-extra-entries (role)
      ```

      ```plaintext Example
      ,giveaway settings remove-extra-entries @member
      ```
    </CodeGroup>

    * **Permissions**: Manage Guild
  </Accordion>

  <Accordion title="Viewing extra entries">
    View all roles configured with extra entries.

    <CodeGroup>
      ```plaintext Syntax
      ,giveaway settings view-extra-entries
      ```

      ```plaintext Example
      ,giveaway settings view-extra-entries
      ```
    </CodeGroup>

    * **Permissions**: Manage Guild
  </Accordion>

  <Accordion title="Ban users from entering giveaways">
    Prevent a user from participating in giveaways.

    <CodeGroup>
      ```plaintext Syntax
      ,giveaway settings ban (user)
      ```

      ```plaintext Example
      ,giveaway settings ban @blahja
      ```
    </CodeGroup>

    * **Permissions**: Moderate Members
  </Accordion>

  <Accordion title="Unban a user from entering">
    Allow a previously banned user to participate in giveaways again.

    <CodeGroup>
      ```plaintext Syntax
      ,giveaway settings unban (user)
      ```

      ```plaintext Example
      ,giveaway settings unban @blahja
      ```
    </CodeGroup>

    * **Permissions**: Moderate Members
  </Accordion>

  <Accordion title="View banned users">
    List all users banned from entering giveaways.

    <CodeGroup>
      ```plaintext Syntax
      ,giveaway settings banned-users
      ```

      ```plaintext Example
      ,giveaway settings banned-users
      ```
    </CodeGroup>

    * **Permissions**: Moderate Members
  </Accordion>

  <Accordion title="View active giveaways">
    Check all currently active giveaways.

    <CodeGroup>
      ```plaintext Syntax
      ,giveaway settings view-giveaways
      ```

      ```plaintext Example
      ,giveaway settings view-giveaways
      ```
    </CodeGroup>

    * **Permissions**: Manage Guild

    * **Aliases**: view-active
  </Accordion>
</AccordionGroup>

## <Icon icon="dice" size={32} /> Rerolling a Giveaway

<CodeGroup>
  ```plaintext Syntax
  ,giveaway reroll (giveaway id)
  ```

  ```plaintext Example
  ,giveaway reroll 1318805763531083800
  ```
</CodeGroup>

## <Icon icon="gear-code" size={32} /> Getting the Giveaway ID

Obtain the message ID by enabling Developer Mode in Discord, right-clicking on the message, and selecting “Copy Message ID”.
