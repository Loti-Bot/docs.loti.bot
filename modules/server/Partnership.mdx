### Description

Manage partnership and affiliate managers, their setup, requirements, weekly posts, and point tracking.

<Note>
Use `,partnership wizard` for a guided step-by-step setup.
</Note>






### `partnership requirement`

<CodeGroup>
  ```plaintext Syntax
  ,partnership requirement
  ```

  ```plaintext Example
  ,partnership requirement
  ```
</CodeGroup>

- Set requirements that posts must meet to count.
- **Aliases**: `req`, `requirements`
- **Arguments**: None
- **Permissions**: None

### `partnership requirement age`

<CodeGroup>
  ```plaintext Syntax
  ,partnership requirement age <duration>
  ```

  ```plaintext Example
  ,partnership requirement age 30d
  ```
</CodeGroup>

- Minimum age of the partnered server. Use `0` to disable.
- **Arguments**: `duration`
- **Permissions**: None

### `partnership requirement members`

<CodeGroup>
  ```plaintext Syntax
  ,partnership requirement members <count>
  ```

  ```plaintext Example
  ,partnership requirement members 100
  ```
</CodeGroup>

- Minimum member count of the partnered server. Use `0` to disable.
- **Arguments**: `count`
- **Permissions**: None
