### Description

Manage ticket categories, panels, staff, logs, and blacklists.

### `tickets send`

<CodeGroup>
	```plaintext Syntax
	,tickets send <channel>
	```

	```plaintext Example
	,tickets send #tickets
	```
</CodeGroup>

- Send the ticket panel in a channel.
- **Arguments**: `channel`
- **Permissions**: Manage Guild
