## Protect your server from bad actors

---

Enable an anti-nuke feature

**Parameters**

- 'role', 'channel', 'emoji', 'ban', 'kick', 'webhook', 'vanity' or 'bot'

**Usage**  
Syntax: `,antinuke enable ('role', 'channel', 'emoji', 'ban', 'kick', 'webhook', 'vanity' or 'bot')`

- **Example**:
  ```
  ,antinuke enable vanity
  ```

---

## Add or remove anti-nuke admins, allowing them to control your anti-nuke configuration.

**Aliases**  
admin, adm

**Usage**  
Syntax: `,antinuke admins`

- **Example**:
  ```
  ,antinuke admins add @user
  ```

---

## Set the action after a threshold has been met for an anti-nuke feature.

**Aliases**  
action, punishment

**Parameters**

- 'ban', 'kick', 'warn' or 'strip', 'role', 'channel', 'emoji', 'ban', 'kick', 'webhook', 'vanity' or 'bot'

**Usage**  
Syntax: `,antinuke do ('ban', 'kick', 'warn' or 'strip') ('role', 'channel', 'emoji', 'ban', 'kick', 'webhook', 'vanity' or 'bot')`

- **Example**:
  ```
  ,antinuke do ban vanity
  ```

## Set the threshold before punishment for an anti-nuke feature

**Aliases**  
amount, amt

**Parameters**

- threshold, 'role', 'channel', 'emoji', 'ban', 'kick', 'webhook', 'vanity' or 'bot'

**Usage**  
Syntax: `,antinuke threshold (threshold) ('role', 'channel', 'emoji', 'ban', 'kick', 'webhook', 'vanity' or 'bot')`

- **Example**:
  ```
  ,antinuke threshold 3 vanity
  ```
