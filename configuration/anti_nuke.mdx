## <Icon size={32} icon="shield" /> `Protect your server from bad actors`

## Simple Setup

`,an enable ban (can be replaced with other types)`

`,an th ban 3 (sets the threshold to 3 for ban)`

`,antinuke p ban kick (kicks the offender once the threshold is hit)`

### Toggle a module in your guild's Anti-Nuke System

<CodeGroup>
  ```plaintext Syntax
  ,antinuke toggle ('Webhook', 'Role', 'Vanity', 'Kick', 'Ban', 'Channels', 'Emojis', 'Stickers' or 'Bot')
  ```

  ```plaintext Example
  ,antinuke toggle webhook
  ```
</CodeGroup>

**Parameters: `webhook, role, vanity, kick, ban, channels, emojis, stickers, bot`**

**Permissions:** Anti Nuke Admin

***

### Remove an administrator from your anti-nuke system

<CodeGroup>
  ```plaintext Syntax

  ```

  ```plaintext Example
  ,antinuke admin delete blahja
  ```
</CodeGroup>

**Parameters: `User to remove`**

### Set a punishment for a module in your guild's Anti-Nuke System

<CodeGroup>
  ```plaintext Syntax
  ,antinuke punishment ('Webhook', 'Role', 'Vanity', 'Kick', 'Ban', 'Channels', 'Emojis', 'Stickers' or 'Bot') ('strip', 'kick' or 'ban')
  ```

  ```plaintext Example
  ,antinuke punishment webhook strip
  ```
</CodeGroup>

**Parameters:**

**1: `webhook, role, vanity, kick, ban, channels, emojis, stickers, bot `**

**2: `strip, kick, ban`**

**Permissions:** Anti Nuke Admin

**Aliases:** `p`

***

### Set a threshold for a module in your guild's Anti-Nuke System
