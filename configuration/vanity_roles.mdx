Reward users with a role when they rep your vanity.

### Note: Assuming `,` is your server prefix.
<Warning>
  Loti must be above the role you want to assign in the role hierarchy.
</Warning>
### Set Vanity
<CodeGroup>
  ```plaintext syntax
  ,vanity set (vanity)
  ```

  ```plaintext example
  ,vanity set /relate
  ```
</CodeGroup>

* **(vanity)**: The vanity URL or keyword.

* **Permissions**: Manage Guild, Manage Roles

### Vanity Channel

<CodeGroup>
  ```plaintext syntax
  ,vanity channel (channel)
  ```

  ```plaintext example
  ,vanity channel #/relate
  ```
</CodeGroup>

* **(channel)**: The channel where the bot will send vanity messages.

* **Permissions**: Manage Guild, Manage Roles

### Vanity Rewards Add

<CodeGroup>
  ```plaintext syntax
  ,vanity rewards add (role)
  ```

  ```plaintext example
  ,vanity rewards add @/relate
  ```
</CodeGroup>

* **(role)**: The role to be assigned, which can be specified as a @role name or a role ID.

* **Permissions**: Administrator

### Vanity Rewards List

<CodeGroup>
  ```plaintext syntax
  ,vanity rewards list
  ```

  ```plaintext example
  ,vanity rewards list
  ```
</CodeGroup>

* Displays all configured vanity rewards.

* **Permissions**: Administrator

### Vanity Rewards Remove

<CodeGroup>
  ```plaintext syntax
  ,vanity rewards remove (role)
  ```

  ```plaintext example
  ,vanity rewards remove @/relate
  ```
</CodeGroup>

* **(role)**: Remove a vanity reward from the existing vanity rewards list.

* **Permissions**: Administrator

### Vanity Message

<CodeGroup>
  ```plaintext syntax
  ,vanity message (message)
  ```

  ```plaintext message example
  ,vanity message {user.mention} has been given {role.mention} for having {vanity} in their status.
  ```

  ```plaintext embed example
  ,vanity message {embed}{title: Thanks For Repping {user.name}}
  ```
</CodeGroup>

* **(message)**: Custom message sent in the vanity channel.

* Can take [variables](./variables.md).

* **Permissions**: Manage Guild, Manage Roles

### Vanity Settings

<CodeGroup>
  ```plaintext syntax
  ,vanity settings
  ```

  ```plaintext example
  ,vanity settings
  ```
</CodeGroup>

* View the vanity configuration in a simple to read format.

* **Permissions**: Manage Guild, Manage Roles

### Vanity Remove

<CodeGroup>
  ```plaintext syntax
  ,vanity remove
  ```

  ```plaintext example
  ,vanity remove
  ```
</CodeGroup>

* Removes the vanity settings for the guild.

* **Permissions**: Manage Guild, Manage Roles
