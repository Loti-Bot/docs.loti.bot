Reward users with a role when they rep your vanity.

### Note: Assuming `,` is your server prefix.

### Set Vanity

,vanity set (vanity) (role)

- **(vanity)**: The vanity URL or keyword.
- **(role)**: The role to be assigned, which can be specified as a @role name or a role ID.

### Vanity Message

,vanity message (message)

- Can take [variables](./variables.md).
- **Example**:
  ```
  ,vanity message {user.mention} has been given {role.mention} for having {vanity} in their status.
  ```

### Set Vanity Channel

,vanity channel (channel)

- Changes the channel where the message is sent when someone reps your vanity.

### View Vanity

,vanity view

- Shows the current vanity set.
