### Description

Configure server settings for Loti.




### `settings action silent`

<CodeGroup>
  ```plaintext Syntax
  ,settings action silent <action>
  ```

  ```plaintext Example
  ,settings action silent ban
  ```
</CodeGroup>

- Toggle whether the bot silently punishes an action without sending a DM.
- **Arguments**: `action`
- **Permissions**: None

### `settings action message`

<CodeGroup>
  ```plaintext Syntax
  ,settings action message <action> <template>
  ```

  ```plaintext Example
  ,settings action message ban {user.mention} was banned for breaking the rules.
  ```
</CodeGroup>

- Set the DM message template for a moderation action.
- **Arguments**: `action`, `template`
- **Permissions**: None
